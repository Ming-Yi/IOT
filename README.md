# 物聯網  
## 利用Java Weka.jar訓練模型
- 訓練集來自本機檔案，     
- 測試集來自於MySql database並修改成正確值
- 在使用HTML+HighChats 呈現

# 檔案說明:
### Java_Code > Bayes 資料夾
- lib  > Jar file  
- data > training data  

### Web_Code 資料夾
- database.php > PHP連接MySql，並且輸出JSON格式的資料
- dataset.php > 使用PHP產生100筆的測試資料
- highchart.html > Html + HighChats 網頁呈現
